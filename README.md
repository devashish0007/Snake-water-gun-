# Snake-water-gun-
A simple game of Snake water gun 

# Rules
The game will consist of 10 rounds.
  YOU VS COMPUTER,
  GUN VS GUN = DRAW,
  GUN VS SNAKE = WIN,
  GUN VS WATER = LOSE,
  WATER VS WATER = DRAW,
  WATER VS GUN = WIN,
  WATER VS SNAKE = LOSE,
  SNAKE VS SNAKE = DRAW,
  SNAKE VS WATER = WIN,
  SNAKE VS GUN = LOSE,
